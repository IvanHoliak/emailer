## Console application on nodejs to search for emails on the site.

1) Clone this repo.
2) `npm i`
3) `npm link`
4) `emailer <site url> <...options>`

#### Options:
| Option      | Description |
| --------------- | --------------- |
| -d, --depth \<number\>| Maximum search depth. (**Default value: 3**)|
